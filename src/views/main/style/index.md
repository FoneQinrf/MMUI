### 内置样式说明
---
#### 内置样式
``` javascript

//文字超出省略
vvm-ellipsis

//上0.5px边框
vvm-hairline-top
//下0.5px边框
vvm-hairline-bottom
//左0.5px边框
vvm-hairline-left
//右0.5px边框
vvm-hairline-right
//上下0.5px边框
vvm-hairline-top-bottom
//全边框0.5px
vvm-hairline-surround

```
#### 内置动画
Vue动画使用文档<a href="https://cn.vuejs.org/v2/api/#transition" target="_blank">transition 组件文档</a>
``` javascript

//fade 淡入淡出
fade

//弹窗动画
//上出
vvm-model-top
//中间弹出
vvm-model-center
//右出
vvm-model-right
//左出
vvm-model-left
//下出
vvm-model-bottom

//旋转loading
vvm-loading

```
